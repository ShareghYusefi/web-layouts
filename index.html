Web Layout

Traditional flow of HTML elements are to be stacked on top of each other. Block elements start on a new line and take up the full width available

Inline elements are those that start on a new line and only take up as much width as necessary.
Span tags are also inline elements which do not take up any width or height value.

Inline-block elements that flow with the text and do NOT start on a new line. They also maintain their size and shape like block elements.

Float css property can be used to move inline elements to the left or right on the page and allow text to wrap around it.
